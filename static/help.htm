How could I see the reports for a given library?

Go to home page, enter the library ID and then click search. You may navigate to different reports using the menus in the header.

How could I download a report?

There are different formats available, .CSV, .XLS, .PDF, .PNG, etc. On the top right of a chart, click the three-lined "Menu" icon and select the format you wish to download.

I entered a correct library ID but I couldn't see any data.

The library may not have recent data available. Refresh your browser and try a different library ID.
